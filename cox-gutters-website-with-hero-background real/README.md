# Cox Gutters Website With Hero Background

This package includes the Cox Gutters homepage with an illustrated roof, gutter, sunny sky, and rooster hero background.

## Files

- `index.html` — GitHub-ready homepage
- `images/cox-gutters-roof-rooster-hero.jpg` — hero background image used by the homepage
- `images/cox-gutters-roof-rooster-hero-preview.jpg` — smaller preview of the background asset

## GitHub upload

Upload both:
- `index.html`
- the full `images` folder

The image path inside the code is:

`images/cox-gutters-roof-rooster-hero.jpg`

If you only replace `index.html` and do not upload the `images` folder, the background will not display.
